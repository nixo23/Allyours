Llaveros

Los llaveros están hechos en madera y cortados a láser, las imágenes al igual son impresas a láser para tener una mejor calidad visual. Ya después de tener la base y la imagen juntas se les agrega una capa de resina para protegerlas de mugre o sustancias.


Libretas

Las libretas es una idea de reciclaje de tela. Se utiliza tela que no ha sido usada en la creación de camisetas y se diseñan a partir de pinturas textiles especializadas para que se quede plasmado el diseño.
